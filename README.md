# RayTracing
RayTracing  

This project is based on [Ray Tracing in One Weekend](https://raytracing.github.io/books/RayTracingInOneWeekend.html)

Installation  
```sh
apt-get update
apt-get install build-essential
```

Compile  
```sh
g++ -o main main.cc
```  

Run  
```sh
./main > image.ppm
```
